2. Obligatoriedad de facilitar los datos y responsabilidad del usuario

Todos los datos personales solicitados, incluida la dirección IP así como los datos de navegación, son obligatorios (salvo que en el campo requerido se especifique lo contrario) para cumplir con las finalidades establecidas, no siendo posible el registro en el Sitio Web sin facilitarnos los datos completos, sin perjuicio de que el usuario podrá visualizar libremente el contenido del Sitio Web.
Asimismo, cuando los datos personales sean recabados a través de formularios que se puedan habilitar en el Sitio Web con motivo de las actividades del mismo, será necesario que el usuario aporte, al menos, aquellos marcados con un asterisco(*), ya que, si no se suministrasen esos datos considerados necesarios, no podremos aceptar y gestionar el servicio web o consulta formulada.
El usuario garantiza que es mayor de catorce (14) años y que los datos aportados son verdaderos, exactos, completos y actualizados, comprometiéndose a notificar a GEOAL cualquier tipo de modificación que se produzca en los datos que haya facilitado.
En todo caso, la contratación de los productos y servicios ofrecidos en el Sitio Web quedará limitada exclusivamente a usuarios mayores de dieciocho (18) años.
Por último, debe tenerse en cuenta que el número de consultas efectuadas por cada usuario a través del Sitio Web podrá limitarse por parte de GEOAL con el objetivo de evitar un número de consultas masivas que pueda suponer una utilización desproporcionada de dicho servicio, suponer un problema de seguridad para la plataforma y el resto de los usuarios o un uso fraudulento de la información y los recursos de GEOAL.
